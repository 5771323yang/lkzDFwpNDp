# Python图书管理系统简介

## 核心功能
- 登录注册
- 图书信息管理（添加、修改、查询、删除）
- 图书借阅记录管理
- 读者信息查询

## 适用场景
- 学校图书馆
- 企业图书馆
- 个人图书管理

## 技术特点
- 基于Flask框架
- 使用MySQL进行数据存储
- 界面采用HTML、CSS和JavaScript技术实现
- 提供运行教程和开发文档
- 包含PPT演示文稿
